My work

File Haven

File Haven allows users to securely store and manage their files and folders. Visitors can explore the interface, but an account is required to upload, organize, or download files. Registration is quick, giving users access to features like creating folders, uploading files, and managing their storage.

File Haven: Rest API

File Haven API is a simple RESTful API for managing File Haven files, folders, and users. It allows a user to perform basic CRUD operations on their files, and folders, and it supports user authentication.

My Blog: Main site

My Blog offers posts on topics like tech, film, and fashion. Visitors can browse and read freely, but an account is required to comment. Registration is quick, allowing users to add and manage their own comments.

My Blog: CMS

My Blog CMS is designed for managing My Blog's content. It allows the blogger to create, edit, and delete posts, and remove comments. Key features include a rich text editor with real-time preview and secure access through authentication.

My Blog: Rest API

My Blog API is a simple RESTful API for managing My Blog posts, comments, tags, and users. It allows a user to perform basic CRUD operations on blog posts, comments, and users, and it supports user authentication.

Resume Builder App

This app offers an intuitive and easy-to-use interface, enabling users to create their resumes quickly and efficiently and with a single click, users can download their completed resume as a PDF file.

Weather App

This weather app offers a simple interface with features such as city search with autocomplete functionality, real-time weather updates, and multi-day forecasts, inspired by the Mac weather app.

My Todo List

My Todo List is a web app to help users organize their tasks. Built with JavaScript for easy management, users can create tasks, organize them into projects, and prioritize them in a simple way.

Tic-Tac-Toe
